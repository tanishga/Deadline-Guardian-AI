# Deadline Guardian AI

An AI-powered productivity assistant that helps students organize tasks, prioritize deadlines, and generate action plans using Google's Gemini API.

## Features

-  Enter any task
-  AI-generated action plans
-  Priority detection
-  Deadline analysis
-  Suggested work schedule
-  Productivity tips

## Built With

- HTML
- CSS
- JavaScript
- Python
- Flask
- Gemini API

## Future Features

- Login system
- Task history
- Calendar integration
- Focus Timer
- Analytics Dashboard
- AI chatbot assistant
- Smart reminders
- Mobile version
